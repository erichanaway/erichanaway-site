Project Overview

The CA-526 Point-in-Time (PIT) Dashboard is an interactive web application designed to make homelessness data easier to explore and understand across Amador, Calaveras, Mariposa, and Tuolumne counties. The dashboard provides visualizations, demographic breakdowns, and summary statistics to support planning and decision-making within the Central Sierra Continuum of Care.

Key Features

County and year filters
Interactive Chart.js visualizations
Demographics and special population views
Data tables for reporting review

Technologies Used

HTML, CSS, JavaScript, Chart.js, SheetJS, Vite, Git, GitHub, and Vercel.

Challenges and Solutions

One of the primary challenges of this project was combining multiple years of PIT data while accounting for differences in reporting structures and field names. Additional effort was required to organize dynamic filters and keep charts synchronized across several dashboard pages.

The application was built using modular JavaScript and Chart.js, allowing charts and KPIs to update automatically based on county and year selections.
